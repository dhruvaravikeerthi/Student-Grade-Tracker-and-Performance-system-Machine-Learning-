Student Performance Prediction System (Machine Learning)

Overview
This project is a machine learning-based web application that predicts student academic performance using both academic records and lifestyle factors. It focuses on showing not just predictions, but also the reasoning behind them, making the results easier to understand.
The system explores how habits like study time, attendance, sleep patterns, and daily routines influence academic outcomes, and presents these insights in a clear and interactive way.

Objectives
1)Build a model to predict student performance
2)Create an easy and interactive web application
3)Show explanations behind predictions
4)Identify key factors affecting student results
5)Make machine learning more understandable and practical



Features
1)Simple Streamlit web interface
2)Real-time prediction system
3)Explainable results with feature impact
4)Model evaluation metrics
5)Visual charts for feature importance
6)Dataset upload for bulk predictions



Machine Learning Approach
The system uses supervised learning trained on student data. It learns patterns between academic/lifestyle inputs and performance outcomes, then uses those patterns to predict results for new inputs.
It is designed to be more interpretable, so users can understand what influences the prediction instead of just seeing an output.

Tech Stack
1)Python
2)Streamlit
3)Pandas & NumPy
4)Scikit-learn
5)Matplotlib / Seaborn

Goal
To show how machine learning can be used in education in a simple, transparent, and meaningful way — helping students and teachers understand performance instead of just predicting it.
