# 🧠 Student Performance Classifier (ML Project)

A Machine Learning web application that predicts student performance using academic and lifestyle features.

---

## 🚀 Features

- Predict student performance using ML model
- Real-time input-based predictions
- Clean Streamlit interface
- Feature contribution analysis
- Visual explanations of model output

---

## 🧠 Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

---

## 📊 Model

- Algorithm: Logistic Regression / Classification Model
- Trained on student performance dataset
- Evaluates factors like study hours, attendance, sleep, etc.

---

## 🎯 Goal

To make student performance prediction **transparent, explainable, and interactive**.
