# 🎯 AI Academic Performance Optimizer

A machine learning-powered web application that predicts student academic performance based on behavioral habits and provides personalized optimization strategies.

---

## 🚀 Overview

This project analyzes how daily student behaviors such as study habits, sleep patterns, stress levels, and social media usage impact academic performance.

It not only predicts scores but also explains:
- why a score was predicted
- which habits influence performance the most
- what changes can improve outcomes

---

## 🧠 Key Features

### 📊 Machine Learning Prediction
- Uses a Random Forest Regressor
- Trained on synthetic behavioral dataset
- Predicts academic performance score (0–100)

### 📈 Behavioral Analysis
- Study hours
- Sleep patterns
- Attendance
- Social media usage
- Stress levels

### 🔍 Explainable AI Insights
- Feature impact visualization
- Strengths and weaknesses breakdown
- Behavioral interpretation of results

### 🔮 What-If Simulation
- Shows how score changes if habits improve
- Compares multiple improvement scenarios

### 🚀 Optimization Engine
- Suggests the single most impactful habit change
- Provides expected score gain

---

## 📊 How It Works

1. User enters academic behavior inputs
2. Model predicts performance score
3. System analyzes contributing factors
4. What-if engine simulates improvements
5. Best action is recommended
